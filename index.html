Welcome to my portfolio

The showcased projects are based on a dataset detailing global layoffs during the COVID-19 pandemic.


I completed these projects and this website using skills I acquired independently through online resources.

Data Cleaning in SQL

In this project, I used MySQL to clean data on global layoffs during COVID-19, from March 2020 to March 2023, as preparatory work for further exploration.

Data exploration

Building on the preparatory cleaning work, I explored the data to extract insights about the companies and industries most impacted during COVID-19. I also used MySQL for this project.

Data visualization

The final step of this project has been to create a visualization in Power BI to make sense of the data. I put together several charts to highlight the key findings uncovered in the exploration. I linked both a PDF and the pbix file in the GitHub folder. The PDF is retrievable directly on GitHub, but you will need to download the pbix to consult it.
